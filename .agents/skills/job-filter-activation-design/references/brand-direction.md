# Editorial Credibility

## Goal

Make Job Filter feel like a premium, truthful, document-first workspace.

## Principles

- Warm-neutral, not cold slate SaaS
- Premium, not flashy
- Calm confidence, not “AI magic”
- Document workflow, not dashboard clutter
- Fewer boxes, more hierarchy through spacing and type

## Visual Rules

- Use forest-led primary accents and warm neutrals
- Use copper as a secondary accent only for emphasis
- Use serif only for selective editorial moments
- Keep controls and dense content in sans-serif
- Prefer large spacing and strong sectioning
- Make the preview feel like a real artifact on paper

## Anti-Patterns

- Purple-forward startup look
- Default dashboard card stacks
- Overdecorated gradients
- Freeform design-tool affordances
- Hiding work behind black-box loaders
