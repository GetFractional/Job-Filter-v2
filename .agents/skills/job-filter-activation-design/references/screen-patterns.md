# Screen Patterns

## Shared Profile Workspace

Three-zone layout:

1. Left rail
   - steps
   - completion
   - issue queue when relevant
2. Main editor
   - current form or review step
3. Right preview
   - live resume preview
   - visible progress during import/build

## Activation Sequence

Top level:

1. Intro
2. Details
3. Experience
4. Skills
5. Extras
6. Preferences

Experience substeps:

1. Import or manual start
2. Confirm companies
3. Confirm roles and dates
4. Confirm responsibilities
5. Confirm results
6. Review skills and tools

## Key UX Rules

- Inline “No resume? Start manually”
- Separate responsibilities from results
- Make summary cards actionable
- Compare options must show visibly different structures
- Keep current-role handling explicit
