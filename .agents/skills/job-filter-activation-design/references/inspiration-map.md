# Inspiration Map

## Borrow

- Step-by-step editor structure from polished onboarding flows
- Live preview pane beside the editor
- Visible build/progress motion during import
- Modular resume-section concept
- Strong top-level progress navigation

## Do Not Borrow

- Full freeform drag-and-drop design-tool behavior
- Decorative consumer-app gradients
- Overly playful brand language
- Template-first flows that hide the canonical profile

## Product Translation

- Use inspiration to improve clarity, pacing, and confidence
- Translate it into Job Filter’s truth-first, proof-backed workflow
- Keep the user as the hero, but let the mechanism stay visible
