# Extraction Decision Rules

## Candidate Selection

Auto-apply only when the best candidate is clearly superior.

Prefer manual compare when:

- confidence gap is small
- suspicious labels are present
- company-role-date structures disagree

## Preview Rules

Preview cards should show:

- top company-role-date triplets
- suspicious/problem notes
- meaningful differences, not generic snippets

## UX Rule

Users should feel that the system is checking multiple possible readings, not failing.
